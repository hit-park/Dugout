# 사회인 야구 경쟁 앱 분석 리포트

**작성일**: 2026.03.19  
**관련 문서**: PRD v1.0, TDD v1.0

---

## 1. 시장 개요

- **등록팀**: 약 3만 팀 (비공식 포함 추정)
- **등록 선수**: 50만 명 이상
- **야구장**: 전국 약 359곳 (면수 기준 500면+, KBO 권장 700면 이상)
- **리그 구조**: 1~4부 리그, 지역 협회 단위 운영
- **특징**: 한 선수가 여러 팀에 소속 가능, 주말 중심 활동

### 핵심 Pain Point (시장 기회)
1. 야구장 부족 → 구장 확보가 팀 운영의 최대 과제
2. 기록 관리 독점 → 게임원 과금 논란으로 대안 수요 증가
3. 팀 운영 도구 부재 → 밴드/카톡에 의존, 체계적 관리 불가
4. 매칭 비효율 → 게시판 기반 수동 매칭, 실력 불균형 빈번

## 2. 주요 경쟁 서비스

### 2-1. 게임원 (GAMEONE) — 시장 지배자

- **운영사**: 주식회사 스탯앤코 (서울 송파구, 2004년 설립)
- **포지셔닝**: 사회인야구 기록 플랫폼

**주요 기능:**
- 경기 기록 관리 (시즌/통산 기록, 박스스코어)
- 일정 및 결과 확인
- 팀 생성, 선수 찾기, 팀 찾기
- 리그 가입 및 리그 내 기록/랭킹
- 커뮤니티 (사회인야구/프로야구 이야기, 벼룩시장)
- 야구장 부킹, 연습장 부킹
- 용병/기록원/심판 모집
- 통합랭킹, 스코어보드, 통합기네스
- 쇼핑 (야구용품 — 원샵)

**강점:**
- 압도적인 사용자 수 (3만 팀, 50만 선수)
- 대부분의 리그가 게임원을 사용 (사실상 표준)
- 오래된 데이터 축적 (2004년~)
- 앱 + 웹 모두 제공

**약점 (= Dugout의 기회):**
- 2022~2023년 과금 방식 변경으로 큰 논란 (팀당 10만원 등)
- 앱스토어 리뷰에서 오류, 느린 속도, UI 불편 불만 다수
- 광고 과다 (최대 20초 광고, 스킵 불가)
- 독과점 구조에 대한 사용자 반감 확산
- **AI 기능 전무** — 단순 기록 조회/입력 중심
- **팀 운영 도구 없음** — 출석 관리, 라인업 추천, 회비 관리 미제공

### 2-2. 스포비 (SPOBEE) — 협회 중심 B2B

- **운영사**: (주)스포비 (경기 시흥)
- **포지셔닝**: 협회/리그 단위 기록관리 시스템 + 영상 서비스

**주요 기능:**
- 협회별 독립 홈페이지 제공 (화이트라벨 방식)
- 기록원 없이 기록 가능한 독립형 기록 시스템
- 경기 영상 촬영/편집 서비스
- AI 기반 영상 편집 R&D (한국공학대학교 LINC+ 연계)

**고객사:** 시흥시, 대전, 송파구, 세종시, 보령시, 영천시, 인제군, 완주군 등 지역 야구협회 다수

**강점:**
- 게임원 대비 합리적인 가격
- 협회 단위 맞춤 서비스
- 기록원 없이 자체 기록 가능
- 영상 서비스 특화

**약점:**
- B2B 모델이라 일반 선수 인지도 낮음
- 개인 사용자 직접 접근 어려움
- 모바일 앱 미제공
- AI 기능은 영상 편집 R&D 단계

### 2-3. 사야인 (sayain.gg) — 신규 진입자

- **포지셔닝**: 사회인야구 통합 포털

**주요 기능:**
- 경기 일정/기록/순위/하이라이트 확인
- 간편 로그인으로 선수 관리
- 매칭센터 — 용병 매칭 (지역/포지션 선택)
- 무료 서비스

**강점:**
- 무료 진입
- 용병 매칭 기능 보유
- 깔끔한 웹 UI

**약점:**
- 사용자 수 아직 적음 (네트워크 효과 부족)
- 모바일 앱 미제공 (웹 기반)
- AI 기능 없음
- 팀 운영 도구 없음

### 2-4. 야구인닷컴 (yaguin.com) — 커뮤니티형

- **포지셔닝**: 커뮤니티 + 기록 + 심판 서비스

**주요 기능:**
- 커뮤니티 게시판
- 팀/선수 구인구직
- 지역별 커뮤니티
- 심판 서비스

**강점:** 오래된 커뮤니티, 충성 사용자  
**약점:** 소규모 운영, 앱 미제공, 기능 제한적

## 4. 현재 팀 운영 도구 생태계

사회인 야구팀이 실제로 사용하는 도구 조합:

```
[소통]     네이버 밴드 (투표, 사진 공유, 공지)
           + 카카오톡 단톡방 (실시간 소통, 급한 연락)

[기록]     게임원 (경기 기록, 리그 순위)
           또는 스포비 (협회 소속 시)

[회비]     엑셀/메모장 + 개별 카톡 입금 요청

[라인업]   주장 머릿속 + 메모장 → 카톡 단톡방에 사진 공유

[매칭]     게임원 게시판 / 밴드 / 카페 게시글

[구장]     전화 / 카톡 / 밴드 (구전 정보)
```

**핵심 인사이트:**
> 사회인 야구팀의 운영은 최소 3~4개 도구에 분산되어 있다.
> 하나로 통합해주는 서비스가 없으며, 특히 "팀 운영" 영역은 완전한 공백이다.
